How to use:

· Open vATIS
· Import > \LIXX\Plugins\vATIS - All the files [LIMM, LIRR, LIBB]
· Remember to change the ARPT COND by pressing on the text "ARPT COND"
· It'll open a new popup and then choose the Runway Condition Report
· That's it!
· For further, check https://docs.vatis.clowd.io/ > Documentation
